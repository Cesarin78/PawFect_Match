# 🐾 PawFect Match — Pet Adoption Website

PawFect Match is a multi-page pet adoption website built to connect 
loving families with animals in need. Users can browse available pets, 
learn about the adoption process, and submit an adoption application 
all in one place.

---

## 📋 Pages Included

- **Home Page** — Welcome banner, automatic image slideshow, and customer reviews
- **Find Your Pet** — Browse pets by category (Dogs, Cats, Rabbits, Guinea Pigs, Birds)
- **Pet Profile** — Detailed profile page for each individual pet
- **Adoption Form** — Application form to start the adoption process
- **About Us** — Information about PawFect Match and our mission
- **Contact Us** — Contact form, shelter hours, star rating system, and location info

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, animations, transitions)
- JavaScript (DOM manipulation, dynamic content)
- Git & GitHub (version control)

---

## ✨ Features

- Automatic image slideshow on the homepage cycling through available animals
- Dropdown navigation menu with pet category submenus
- Interactive star rating system on the contact page
- Customer review submission that updates instantly without page reload
- Responsive two-column layout using CSS Flexbox
- Smooth hover animations on buttons and cards
- Consistent blue and orange branding across all pages
- Form success messages handled with JavaScript DOM manipulation

---

## 💡 Purpose

This project was built to practice and demonstrate skills in front-end 
web development including HTML structure, CSS styling, JavaScript 
interactivity, and multi-page site organization.
